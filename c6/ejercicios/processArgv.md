## process.argv: 
Es una propiedad global en Node.js que devuelve un array que contiene los argumentos pasados cuando se lanza el proceso por terminal.
  * process.argv[0]: Es la ruta absoluta del ejecutable de Node.
  * process.argv[1]: Es la ruta absoluta del archivo JavaScript que estamos ejecuta 
  * process.argv[2]: Es el primer argumento real que nosotros pasamos (en este caso, el nombre)0
